# 003 - plcc-scan brief usage message should mention --help

**Type:** fix
**Date:** 2026-05-06

## Description

`plcc-scan`'s brief usage message does not mention that a `--help` option is available for more details. Users who see the brief message have no indication that further help exists.

## Notes

The brief usage message should include a line such as `Run with --help for more information.`
