# 077 - Should the docs include a licensing page?

**Type:** docs
**Date:** 2026-06-07

## Description

The docs do not currently surface licensing information. Users who want to know under what terms they can use, modify, or redistribute plcc-ng have no obvious place to look in the docs.

## Question

Should the docs include a licensing page or section, and if so, what should it contain?

## Notes

Options to consider:

- Add a dedicated "License" page that states the license and links to the full LICENSE file in the repository.
- Add a brief license notice to the home page or footer with a link to the repository.
- Do nothing if the LICENSE file in the repo is considered sufficient.
