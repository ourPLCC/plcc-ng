# 101 - Add an acknowledgments page to the docs

**Type:** docs
**Date:** 2026-06-19

## Description

The docs have no acknowledgments page. There is no place to recognize contributors, institutions, funding sources, or tools that plcc-ng builds on.

## Desired Behavior

Add an acknowledgments page to the docs site that credits:

- Significant contributors and maintainers.
- Any institutions or funding sources that supported development.
- Notable open-source tools or libraries the project depends on (where crediting them is appropriate).

## Notes

Common placements: a standalone page linked from the footer or navigation, or a section at the bottom of the home page. Many doc frameworks (MkDocs Material, Docusaurus) support a footer with links — that is a natural home for an acknowledgments link.
