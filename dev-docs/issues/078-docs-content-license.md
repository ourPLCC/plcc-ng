# 078 - The documentation itself needs a license

**Type:** docs
**Date:** 2026-06-07

## Description

The documentation content (prose, examples, guides) has no stated license. Without one, the legal default is "all rights reserved," which prevents others from reproducing or adapting the docs even for legitimate purposes.

## Desired Behavior

- Choose a license for the documentation content (e.g. Creative Commons CC BY 4.0 is common for open documentation).
- Add a license notice to the docs site (footer or dedicated page) and a corresponding LICENSE file or section covering the docs.

## Notes

The docs license may differ from the software license — that is normal and expected. Resolving this issue and issue 077 together makes sense, since both concern how licensing is communicated in the docs.
