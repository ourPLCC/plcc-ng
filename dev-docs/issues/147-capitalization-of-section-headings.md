# 147 - Capitalization of section headings

**Type:** docs
**Date:** 2026-07-06

## Description

Capitalization of section headings in the documentation is not
consistent. It appears to be either (1) capitalize only the first
word except if the word is a proper name or (2) capitalize every
word except for conjunctions, articles, two-letter prepositions, and
the word "for".

No strong preference either way, but consistency is the goal.
Option (1) is being used for the textbook for now, but that's open to
change.
