# 076 - Should the home page have an "Install" section if the quickstart already covers it?

**Type:** docs
**Date:** 2026-06-07

## Description

The home page currently includes install instructions (or a link to them), but the quickstart is about to cover installation as its first step (see issue 074). Having both may create redundancy or confusion about which path a new user should follow.

## Question

Should the home page install content be removed, kept as a standalone shortcut, or folded entirely into the quickstart?

## Notes

Options to consider:

- Remove install from the home page and let the quickstart be the single entry point.
- Keep a minimal "install in one line" on the home page as a quick reference, with the quickstart for first-time users.
- Keep the dedicated install page but remove any duplicate content from the home page.
