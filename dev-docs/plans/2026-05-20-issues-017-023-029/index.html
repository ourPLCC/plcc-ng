    Issues 017, 023, 029 Implementation Plan

    For agentic workers: REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (- [ ]) syntax for tracking.

    Goal: Fix three independent diagnostic issues: include character and position in the plcc-tokens unrecognized-character error (017), annotate empty productions in plcc-parse tree output (023), and exit cleanly on ^C in plcc-rep (029).

    Architecture: Each fix is a single-file change confined to one logical function. They share no state and can be implemented in any order. All follow the project TDD inner loop: failing test → minimal implementation → passing test → commit.

    Tech Stack: Python, pytest (bin/test/units.bash), git worktree branch docs/issues-029-023-017 at .worktrees/docs-issues-029-023-017/

    All commands below must be run from .worktrees/docs-issues-029-023-017/.
